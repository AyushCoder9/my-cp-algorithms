## 🛠 How to Use

To run any of the algorithms locally on your machine, just clone the repo and execute the respective Python file:

```bash
# Clone the repository
git clone https://github.com/AyushCoder9/my-cp-algorithms.git

# Navigate to the correct directory
cd my-cp-algorithms

# Run any python script
python3 <filename>.py
```

_Note: Some algorithms come with pre-populated dummy data and `print` statements to demonstrate their basic functionality when run directly._
